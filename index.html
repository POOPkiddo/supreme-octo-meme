Arduino RFID Door Lock

What The Heck Does A RFID Door Lock Even D0?

It locks a door..., what do you expect??? Let us say we have a annoying sibling as a sibling. With this we could LOCK our door, so they wouldnt be able to get in(hopefully)

How Does It WORK?

The RC522 module is connected to the main board. When a card with a UID added to the code is tapped against the module, it tells the main board to start a motor to unlock a board.
